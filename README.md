## 项目介绍

此项目是[pywxrobot2.0](https://github.com/kanadeblisst00/pywxrobot2.0)的升级版本，支持微信`4.1.1.19`，后面看情况更新到最新版本。目前项目还处于逆向阶段，功能更新会比较缓慢。

现在所有的插件都可以免费使用，如果想定制插件的可提供需求(需要的人多的话也会实现)。

#### 数据采集需求

**需要公众号历史采集/搜一搜/视频号等数据可以看这个接口: https://wqup3673wn.apifox.cn/**

#### 二次开发接口

**目前给的插件都是开箱即用的，如果有需要进行二次开发的，可以看考虑带HTTP接口的插件：https://sn2rohg101.apifox.cn**

#### 下载地址

`https://www.123865.com/s/ihEKVv-Sflx?pwd=vCI5#`

微信的话使用安装版(exe)或者绿色版(7z,zip)都可以，后者解压之后就能用，可以和其他版本共存。

## 使用教程

#### 准备工作

为了防止项目被倒卖，设置了加入交流群才能使用，如果你想在多个账号下使用，你可以自己邀请朋友或其他小号加入群聊(这个是不限制的)

#### 开始使用

先从网盘下载好指定的微信版本和你需要的插件，然后启动并登录微信，再打开插件按提示操作即可

#### 常见问题总结

请参阅[FAQ.md](./FAQ.md)获取常见问题及解答。
