## 项目介绍

此项目是[pywxrobot2.0](https://github.com/kanadeblisst00/pywxrobot2.0)的升级版本，支持微信`4.1.1.19`，后面看情况更新到最新版本。目前项目还处于逆向阶段，功能更新会比较缓慢。

现在所有的插件都可以免费使用，如果想定制插件的可提供需求(需要的人多的话也会实现)。

## 使用教程

#### 准备工作

为了防止项目被倒卖，设置了加入交流群才能使用，如果你想在多个账号下使用，你可以自己邀请朋友或其他小号加入群聊(这个是不限制的)

#### 开始使用

先从网盘下载好指定的微信版本和你需要的插件，然后启动并登录微信，再打开插件按提示操作即可
